# wastewise
Its a waste management software that recogonizes waste by scanning an image .

So you can use this tool when you want to scan an image for three types of garbadge bins and find which garbadge bin to put this in .
- General waste
- Recycling
- Organic

This tool is free and open source to use it . Clone or download this git repo and go into the directory and run _npm start_ to start the web server and goto http://localhost:8080 to acess the web interface or go to http://127.0.0.1:8080 if the previous link doesnt work . Feel free to do anything with the code .
